AYO BELAJAR HTML "DOM & jQuery"

1. Apa Itu jQuery?
jQuery adalah pustaka JavaScript yang cepat, kecil, dan kaya fitur. Ini membuat manipulasi dokumen HTML, penanganan event, dan animasi menjadi jauh lebih sederhana.
2. Apa itu DOM ?
DOM (Document Object Model) adalah jembatan antara kode JavaScript dan tampilan HTML. Tanpa DOM, website kita akan statis dan tidak bisa berinteraksi dengan user.
3. Apakah jQuery masih relevan di 2025?
Meskipun framework modern seperti React populer, jQuery masih digunakan di jutaan website lama (legacy) dan sangat bagus untuk pemula memahami konsep manipulasi elemen dengan mudah.
4. Bagaimana cara menyembunyikan elemen di jQuery?
Kita bisa menggunakan method .hide(), .fadeOut(), atau .slideUp(). Untuk memunculkan kembali, gunakan kebalikannya atau .toggle().
5. Apa bedanya ID dan Class?
ID (#) bersifat unik dan hanya boleh ada satu dalam satu halaman. Class (.) bisa digunakan berulang kali pada banyak elemen sekaligus.
